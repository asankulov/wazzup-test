## Description
#### Wazzup Test Project with Node.js, Express.js and Postgres

## Prerequisites
* Installed Node.js >= 12
* Postgres database

## Installation

```bash
$ npm install
```

## Running the app

1. edit `.env` file from example by adding database credentials and so on

```bash

# development
$ npm run start
```

## Test

```bash
# unit tests
$ npm run test
```
